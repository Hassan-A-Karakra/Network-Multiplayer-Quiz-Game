Chapter 1 Topic: What is the Internet?

The Internet is a global network of interconnected devices that communicate using standard protocols. It consists of end systems (hosts), routers, and communication links.

Main Components:

  • End Systems: Laptops, Phones, Servers
  • Routers: Devices forwarding data
  • Protocols: TCP/IP, HTTP, DNS

Key Functions:

  1. Data Transmission
  2. Routing & Addressing
  3. Error Detection & Correction
